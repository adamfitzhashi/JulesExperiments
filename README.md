# JulesExperiments
A repo to play with the Google tool Jules (https://jules.google.com)

## Bardic Inspiration Macro

This is a macro for Roll20 that helps a D&D 5e Bard player use their Bardic Inspiration feature. It provides a prompt to select a target, choose an inspirational phrase, and select the bard's level to determine the correct inspiration die.

The macro and its documentation can be found in [bardic_inspiration_macro.md](bardic_inspiration_macro.md).
